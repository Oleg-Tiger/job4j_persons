persons
===========

## Описание
Учебный проект курса job4j persons. Реализация REST сервиса.
В данном проекте подключена сборка через Maven.

## Используемые технологии:
- JDK17
- Spring 2.7.12
- Maven 3.8.1
- PostgreSQL
- Lombok

## Требуемое окружение:
- JDK17
- Maven 3.8.1
- PostgreSQL 13.3
